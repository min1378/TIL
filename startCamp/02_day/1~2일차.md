   bs4.BeatifulSoup(문서) 이쁘게 정렬

   .select(selector)

   문서안에 있는 특정 내용을 뽑아줘

   .select_one(selector)

   문서 안에 있는 특정 내용을 하나만 뽑아줘

   ```python
   import requests #요청
   
   import bs4 #이쁘게 파싱
   
   url = 'https://www.naver.com'
   
   selector = '.ah_k'
   
   
   
   html=requests.get(url).text
   
   
   
   soup =bs4.BeautifulSoup(html,'html.parser')
   
   ranks=soup.select(selector)
   
   for ranks in ranks:
   
   ​    print(ranks.text)
   ```

   작업한파일    ->    커밋할 목록    ->     쌓인 커밋들 -> github
